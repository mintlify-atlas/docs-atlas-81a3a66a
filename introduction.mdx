# Welcome to marco-code

marco-code is a minimal LLM-powered coding assistant that demonstrates the core "agent loop" pattern used by AI coding assistants. The assistant iteratively calls tools until it has enough information to respond to your prompts.

## What You Can Do

marco-code helps you with software engineering tasks by:

- Reading and analyzing files in your codebase
- Writing and modifying code files
- Executing shell commands for testing and validation
- Iterating until your task is complete

<Note>
marco-code uses the OpenRouter API to access various LLM models. You'll need an API key to get started.
</Note>
